A sequence is a special kind of function which has domain equal to the natural numbers ℕ={1,2,3,…}\mathbb{N} = \{1,2,3,\ldots\}. We say that an interval (a,b)(a,b) eventually contains a sequence s(n)s(n) if there is a number NN such that a<s(n)<ba < s(n) < b for all n≥Nn \ge N. If every interval (a,b)(a,b) that contains LL eventually contains s(n)s(n), then we say that s(n)s(n) converges to the limit LL. This can be written as: limn→∞s(n)=L.\lim_{n \rightarrow \infty} s(n) = L.

We finished by looking at another example of a limit. Galileo was the first person to observe that the distance traveled by an object that is dropped from a great height is roughly d=4.9t2d = 4.9t^2 meters (when tt is the time in seconds after the object was dropped). The average velocity of a falling object is vaverage=d(t2)−d(t1)t2−t1.v_{\text{average}} = \frac{d(t_2) - d(t_1)}{t_2 - t_1}.

  1. Use Desmos to find the average velocity over the following time intervals:

Mon, Jan 29

Today we defined limits for functions. We say that the limit of f(x)f(x) as xx approaches aa is LL and write limx→af(x)=L\lim_{x \rightarrow a} f(x) = L if every sequence x(n)x(n) that converges to aa (but never equals aa) has limn→∞f(x(n))=L\lim_{n \rightarrow \infty} f(x(n)) = L.

We looked at two examples on Desmos where a function is not defined but its limit is:

  1. limx→1x3−1x−1\lim_{x \rightarrow 1} \dfrac{x^3 - 1}{x-1}.

  2. limx→0x2+9−3x2\lim_{x \rightarrow 0} \dfrac{\sqrt{x^2 + 9} - 3}{x^2}.

Both of these are examples of hole discontinuities. We also saw three other common types of discontinuities:

  1. (Pole discontinuity) f(x)=1xf(x) = \dfrac{1}{x} at x=0x = 0.

  2. (Jump discontinuity) sign(x)={|x|x if x≠00 if x=0\operatorname{sign}(x) = \begin{cases} \dfrac{|x|}{x} & \text{ if } x \ne 0 \\ 0 & \text{ if } x = 0 \end{cases}.

  3. (Oscillation discontinuity) f(x)=sin(1/x)f(x) = \sin(1/x) at x=0x=0.

In all three of these cases, the limit at the point of interest does not exist. But there are also one-sided limits which do exist. For example, the one sided limit as xx approaches 00 from above is in example 3 is: limx→0+1x=+∞\lim_{x \rightarrow 0^+} \dfrac{1}{x} = +\infty and the one-sided limit as xx approaches 00 from below in number 4 is: limx→0−sign(x)=−1.\lim_{x \rightarrow 0^-} \operatorname{sign}(x) = -1. You mostly just need an intuitive understanding that a limit is the yy-value that the graph is heading towards, not the actual yy-value at the point.

At this point, defined continuous functions. A function f(x)f(x) is continuous at x=ax = a if limx→af(x)=f(a)\lim_{x \rightarrow a} f(x) = f(a). Intuitively, a function is continuous if you can draw it without lifting your pencil. Most simple functions (including all linear functions and sin(x)\sin(x) and cos(x)\cos(x)) are continuous at every real number. It turns out that every function constructed from other continuous functions using the operations of addition, subtraction, multiplication, division, powers, and function composition are always continuous at every point in their domains. The only functions you need to worry about are piecewise functions.

  1. limx→4f(x)\lim_{x \rightarrow 4} f(x) where f(x)={x+2x−1 for x≤4x for x>4.f(x) = \begin{cases} \frac{x+2}{x-1} & \text{ for } x \le 4 \\ \sqrt{x} & \text{ for } x > 4.\end{cases} (https://youtu.be/2xdh0yKopB8)

Today we observed that the derivative of f(x)f(x) is a function f′(a)f'(a) that depends on the point aa where you found the slope of the tangent line. Another way to write the definition of derivative is: f′(x)=limh→0f(x+h)−f(x)h.f'(x) = \lim_{h \rightarrow 0} \frac{f(x+h) - f(x)}{h}. We also have two different notations for the derivative function. We’ve already seen f′(x)f'(x). We also use the symbol ddx\dfrac{d}{dx} as a command that literally means “take the derivative of” whatever function comes next. So f′(x)f'(x) means the same thing as ddxf(x).\dfrac{d}{dx} \, f(x). Another notation that we use frequently is to write dydx\dfrac{dy}{dx} to represent the derivative of a function y=f(x)y = f(x). So if we have a function y=f(x)y = f(x), then all of these are the same: f′(x)=ddxf(x)=dydx.f'(x) = \dfrac{d}{dx} \, f(x) = \dfrac{dy}{dx}.

Today we introduced the following derivative rules.

  1. Linear Function Rule. ddx[mx+b]=m\dfrac{d}{dx} [m x + b] = m.
  2. Addition Rule. ddx[f(x)+g(x)]=f′(x)+g′(x)\displaystyle\dfrac{d}{dx} \left[f(x) + g(x) \right] = f'(x) + g'(x).
  3. Constant Multiple Rule. ddx[cf(x)]=cf′(x)\dfrac{d}{dx} \left[ c f(x) \right] = c f'(x).
  4. Power Rule. ddxxn=nxn−1\dfrac{d}{dx} x^n = n x^{n-1}.
