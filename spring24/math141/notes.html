Today we defined limits for functions. We say that the limit of f(x)f(x) as xx approaches aa is LL and write limx→af(x)=L\lim_{x \rightarrow a} f(x) = L if every sequence x(n)x(n) that converges to aa (but never equals aa) has limn→∞f(x(n))=L\lim_{n \rightarrow \infty} f(x(n)) = L.

We looked at two examples on Desmos where a function is not defined but its limit is:

  1. limx→1x3−1x−1\lim_{x \rightarrow 1} \dfrac{x^3 - 1}{x-1}.

  2. limx→0x2+9−3x2\lim_{x \rightarrow 0} \dfrac{\sqrt{x^2 + 9} - 3}{x^2}.

Both of these are examples of hole discontinuities. We also saw three other common types of discontinuities:

  1. (Pole discontinuity) f(x)=1xf(x) = \dfrac{1}{x} at x=0x = 0.

  2. (Jump discontinuity) sign(x)={|x|x if x≠00 if x=0\operatorname{sign}(x) = \begin{cases} \dfrac{|x|}{x} & \text{ if } x \ne 0 \\ 0 & \text{ if } x = 0 \end{cases}.

  3. (Oscillation discontinuity) f(x)=sin(1/x)f(x) = \sin(1/x) at x=0x=0.

In all three of these cases, the limit at the point of interest does not exist. But there are also one-sided limits which do exist. For example, the one sided limit as xx approaches 00 from above is in example 3 is: limx→0+1x=+∞\lim_{x \rightarrow 0^+} \dfrac{1}{x} = +\infty and the one-sided limit as xx approaches 00 from below in number 4 is: limx→0−sign(x)=−1.\lim_{x \rightarrow 0^-} \operatorname{sign}(x) = -1. You mostly just need an intuitive understanding that a limit is the yy-value that the graph is heading towards, not the actual yy-value at the point.

Implicit equations involving xx and yy are sometimes called implicit functions, but they aren’t really functions because one xx value can correspond to multiple yy values. We started with this example.

Rolle’s theorem. If ff is a continuous function on an interval [a,b][a,b], and ff is differentiable on (a,b)(a,b), and f(a)=f(b)f(a) = f(b), then there must be a point cc such that a<c<ba < c < b and f′(c)=0f'(c) = 0.
