Wed, Jan 17

Today we reviewed functions and function notation. We talked about how f(x)f(x) looks like ff multiplied by xx, but it really means something completely different! We also talked about the following important functions (and their graphs) that you should have memorized:

  1. Linear functions f(x)=mx+bf(x) = mx+b.

  2. The square function f(x)=x2f(x) = x^2.

  3. The positive square-root function f(x)=x=x1/2f(x) = \sqrt{x} = x^{1/2}.

  4. The reciprocal function f(x)=1xf(x) = \dfrac{1}{x}.

  5. The 1-dimensional distance function f(x)=|x−a|f(x) = |x-a|.

We reviewed the definitions of domain, range, and function composition. We did the following examples in class.

  1. Let f(x)=4x+2f(x) = \dfrac{4}{x+2} and g(x)=1xg(x) = \dfrac{1}{x}. Find the domain of f(g(x))f(g(x)). (https://youtu.be/PNzHrPebKOw)

  2. Two poles are connected by a wire that also connects to the ground between the poles. Find a formula for the length of the wire as a function of xx and determine the domain.

We didn’t have time for this exercise, but it is also a good example of how to construct a function.
