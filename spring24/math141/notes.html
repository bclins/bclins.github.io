Fri, Jan 26

Today we started talking about limits. We began with this example:

  1. What is the area of a regular nn-gon inscribed in the unit circle?

  2. What happens to the area as the number of sides gets bigger and bigger?

A sequence is a special kind of function which has domain equal to the natural numbers ℕ={1,2,3,…}\mathbb{N} = \{1,2,3,\ldots\}. We say that an interval (a,b)(a,b) eventually contains a sequence s(n)s(n) if there is a number NN such that a<s(n)<ba < s(n) < b for all n≥Nn \ge N. If every interval (a,b)(a,b) that contains LL eventually contains s(n)s(n), then we say that s(n)s(n) converges to the limit LL. This can be written as: limn→∞s(n)=L.\lim_{n \rightarrow \infty} s(n) = L.

We finished by looking at another example of a limit. Galileo was the first person to observe that the distance traveled by an object that is dropped from a great height is roughly d=4.9t2d = 4.9t^2 meters (when tt is the time in seconds after the object was dropped). The average velocity of a falling object is vaverage=d(t2)−d(t1)t2−t1.v_{\text{average}} = \frac{d(t_2) - d(t_1)}{t_2 - t_1}.

  1. Use Desmos to find the average velocity over the following time intervals:
    1. [2,3][2,3]
    2. [2,2.1][2,2.1]
    3. [2,2.01][2,2.01]
    4. [2,2.001][2,2.001]
  2. Use the previous answers to estimate the velocity at the instant when the object has been falling for 2 seconds.

These two examples, the area of a circle and the instantaneous velocity of a falling object, are both limits. Next week, we’ll look at how to work systematically with limits.
