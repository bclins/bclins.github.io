\question Consider the non-homogeneous linear system
$$\frac{d \mathbf{y}}{dt} =   A \mathbf{y}  + \mathbf{b} , \text{ where } A = \begin{bmatrix} 1 & 3 \\ -1 & 1 \end{bmatrix}\text{ and } \mathbf{b} = \begin{bmatrix} -6 \\ 2 \end{bmatrix}.$$
\begin{parts}
\item Find the equilibrium point for the system.  
\vfill

\item Find the general solution to the system above.  Hint, the eigenvectors and eigenvalues of $A$ are $\lambda_1 = 1 - \sqrt{3} i$ with $\mathbf{v}_1 = \begin{bmatrix}\sqrt{3} i\\1 \end{bmatrix}$ and $\lambda_2 = 1 + \sqrt{3} i$ with $\mathbf{v}_2 = \begin{bmatrix}-\sqrt{3} i\\1 \end{bmatrix}$.
\vfill

\item If we replace the constant vector $\mathbf{b}$ above with the vector-valued function $\mathbf{b}(t) = \begin{bmatrix} -6e^{-t} \\ 2 e^{-t} \end{bmatrix}$, then what would make a good guess for a particular solution $\mathbf{y}_p$ for this new system? 
\vfill
\end{parts}

\question Consider the following forced harmonic oscillator.  
  $$y'' + 3y' + 2y = \cos 2t$$
\begin{parts}
\item What is the complexification of this differential equation?  
\vfill

\item Find a particular solution for this differential equation.  
\vfill

\item Find the general solution for this differential equation.
\vfill

\item If this system represents a harmonic oscillator, is it overdamped or underdamped?  How can you tell?  
\vfill
\end{parts}

\question Consider the system 
$$\dfrac{dx}{dt} = x \cos(xy)$$
$$\dfrac{dy}{dt} = -y \cos(xy)$$
\begin{parts}
\item Show that the quantity $H(x,y) = \sin(xy)$ is a Hamiltonian function for the system.
\vfill

\item Sketch the level sets for this system.  Hint: The level sets occur when $xy$ is a constant.  What does the graph of $xy = c$ look like?  
\vfill

\item Sketch the solution curve that passes through the point $(1,\pi)$.  Be sure to indicate the direction of the solution.
\vfill

\end{parts}
